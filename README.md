# Blog Website

Welcome to the blog website repository! This is a simple blog site where users can browse through blog posts and read various articles.

## Features
- Simple, clean layout for displaying blog posts.
- Responsive navigation menu.
- Each blog post contains a title and a short description.
- Fixed footer displaying copyright information.

## Technologies Used
- HTML
- CSS
